Event Class¶

class simulation.Event(packet, EventHandler, EventType, EventTime, flow)¶

Events are enqueued into the Simulator priority queue by their time. Events have a type (PUT, SEND, RECEIVE, GENERATEACK, GENERATEPACK) describing what is done to the packet. Each type of event has an associated network handler (Link, Device, Flow, respectively).



Simluator Class¶

class simulation.Simulator(network, TCP_type, metric)¶
done()¶

Called when the simulation is finished. If metrics were recorded, this closes the files that log data for the useful metrics (e.g. link rate), so the file ends with EOF.

insertEvent(event)¶

This will insert an event into the Priority Queue.

Parameters:event (Event) – This is the event we’re adding into the queue.
processEvent()¶

Pops and processes event from queue.
